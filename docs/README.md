## 🎯 What is MultiCode?

MultiCode is a professional multi-agent AI coding assistant that uses a dynamic debate system to produce secure, high-quality code. It creates custom teams of AI agents that collaborate like a real software development team.

### Key Features

- **Multi-Agent Collaboration**: Multiple AI agents with specialized roles
- **Dynamic Workflows**: AI-generated workflows based on task complexity
- **Secure Execution**: Sandboxed file and shell operations
- **Model Flexibility**: Access to 300+ AI models via OpenRouter
